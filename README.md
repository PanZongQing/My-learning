# 项目说明

### 项目结构
- conf：用于存储配置文件
- middleware：应用中间件
- models：应用数据库模型
- pkg：第三方包
- routers 路由逻辑处理
- runtime 应用运行时数据

## 开发思维
- 建立项目结构路径
- 编写项目配置包，优先定义好配置
- 建立调用配置的 setting模块
- 编写API错误码包
- 编写工具包 
  - 分页页码的获取方式
- 编写models init(包含数据库连接方式等)
- 编写项目启动、路由文件
- 编写业务代码。 
  - 定义接口
  - 编写路由空壳
  - 注册路由
  - 编写标签列表的models逻辑
